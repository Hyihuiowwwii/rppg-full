Model Information

Technical details and specifications of the rPPG heart rate monitoring system.

Model Overview

System: HeartRate Pro v1.0.0

Goal: Real-time remote heart rate monitoring using facial video.

  • Non-contact measurement
  • Real-time processing
  • Dataset validation
  • Web-based interface

Technical Specifications

Face detection: Haar Cascade

ROI: Forehead region

Signal extraction: Green channel mean

Signal processing: Detrending + FFT

Heart rate range: 40 to 180 BPM

Sampling target: ~30 FPS

Video Input
Webcam / Demo video
→
Face Detection
Haar Cascade
→
ROI Extraction
Forehead region
→
Signal Processing
FFT + BPM

Performance

Real-time output: Yes

Buffer target: 150 samples

Average delay: Few seconds

Dataset Validation

Supports UBFC/Kaggle style dataset folders containing:

  • Video file
  • ground_truth.txt

Deep Learning

Model: 1D CNN

Training file: model/train_dl_model.py

Prediction file: model/predict_dl.py
