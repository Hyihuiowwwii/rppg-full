Model Overview

HeartRate Pro v1.0.0

Advanced real-time heart rate monitoring using remote photoplethysmography (rPPG). This non-contact method analyzes subtle color changes in facial skin to detect heart rate.

  • Non-contact measurement
  • Real-time processing
  • Dataset validation
  • Web-based interface
